# 阴阳理论 · 互动学习 WebApp (静态)

纯静态学习/科普 WebApp：

- 动态太极（滑块：阴阳消长 + 极则反提示）
- 分层拆解（表里/动静/物质功能/上下/寒热）
- 四种失衡（阴盛/阳盛/阴虚/阳虚）
- 转化模拟（阈值 + 反转）
- 真假判断训练（真寒假热 / 真热假寒 / 真虚假实 / 真实假虚）
- 术语词典（含中英）

> 注意：本项目不提供医疗诊断或处方建议，仅用于学习理解。

## 部署到 Vercel

1) 推到 GitHub：确保仓库根目录包含 `index.html`  
2) Vercel → Import Project → 选择仓库  
3) Framework 选 **Other**；Build Command 留空；直接 Deploy
